# MTStat

C++で関数型言語のように書くことができるステートパターンライブラリ  
動作環境 : Visual C++    
言語バージョン : C++20  
を想定

## ライブラリの使い方

### 準備
1. リポジトリ内の `/MTStat/MTStat.h` を使いたいプロジェクトに追加する
2. MTStat.h をインクルードするだけ！

### 書き方

- ステート表現用に enum を宣言
- MTStatクラスの型テンプレートに宣言した enum を入れる
- クラスをインスタンスし、メンバ関数 On ~ を呼び、第1引数に対応列挙子,  
  第2引数に void() 型のラムダ式を書く  
  ※メンバ関数 On ~ は繋げて書くことができる。
- インスタンスした変数のメンバ関数 Change を呼び、第1引数に変更したいステート列挙子を書く
- インスタンスした変数のメンバ関数 Update を呼び、現在のステートでの更新処理が呼ばれる

サンプルコードは [TheMain.cpp](https://github.com/3doriTea/MTStat/blob/master/TheMain.cpp) を参照ください。

## 特徴

- ステートパターンを楽に実装できる
- 関数型風に書けることで、コードの文字数を削減できる
